# Spotify Clone - Web Player

A fully responsive music streaming interface built with **JavaScript, HTML5, and CSS3**. This project replicates the Spotify web experience, featuring dynamic playlist rendering and a custom-built audio player.

## 🚀 Features

* **Dynamic Music Library:** Songs and album data are fetched dynamically from JSON metadata, allowing for easy library updates.
* **Custom Audio Controls:** A fully functional playbar with play/pause, next/previous, seekable progress bar, and volume control.
* **Fully Responsive:** Optimized for all screen sizes using CSS Flexbox and Media Queries, including a functional hamburger menu for mobile users.
* **Interactive UI:** Hover effects, smooth transitions, and real-time time/duration updates for a premium user experience.

## 🛠️ Built With

* **HTML5:** Semantic structure and SVG iconography.
* **CSS3:** Custom styling, animations, and responsive layout.
* **JavaScript (ES6+):** Asynchronous data fetching, DOM manipulation, and HTML5 Audio API.

## 📂 Project Structure

* `/Songs`: Contains album folders, audio files, and `info.json` for metadata.
* `Spotify.js`: Core logic for music playback and UI interactions.
* `Spotify.css` & `Utility.css`: Styling and reusable layout classes.

## 🔧 How to Run

1. Clone the repository.
2. Ensure your song folders contain a valid `info.json` and `card.jpg`.
3. Open `index.html` via a local server (like Live Server in VS Code) to allow `fetch` requests to work properly.
